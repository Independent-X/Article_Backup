这里是拿来备份一些社会上我认为不公正的事件或者我反对的文章的。

当我觉得一件事令人震惊，我也会放在这里。

这里一般不会备份的事件有：

1.华为的事件，那些我会单独整理，因为太多了。

2.政府机关(文化部、网信办)的事件，那些我会单独整理，因为对于某一事件来讲，细节类的信息太多了。
  注意：一些政府方面的我认为错误的宣传我也会往这里放——只要未构成事件。
